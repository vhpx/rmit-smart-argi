# @tutur3u/supabase

Supabase client and utilities for Tuturuuu Platform.

## Installation

```bash
npm install @tutur3u/supabase
# or
yarn add @tutur3u/supabase
# or
pnpm add @tutur3u/supabase
```

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test
```
