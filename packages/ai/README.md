# @tutur3u/ai

AI utilities and helpers for Tuturuuu Platform.

## Installation

```bash
npm install @tutur3u/ai
# or
yarn add @tutur3u/ai
# or
pnpm add @tutur3u/ai
```

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test
```
